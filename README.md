# Education Navigator
Education Navigator is a web platform designed to help students in Nepal explore colleges and academic programs in one place. Instead of searching across multiple websites, students can browse institutions, compare programs, view locations on a map, and even build a professional CV directly from the platform.
## Features
- Browse colleges and programs available in Nepal
- Interactive college map using Leaflet.js
- Search and filter colleges by name or location
- Built-in CV Builder to generate a professional CV
- Secure user registration and login
- Personal dashboard for profile management
## Tech Stack
- Backend: Python, Flask, Flask-SQLAlchemy
- Frontend: HTML, CSS, JavaScript
- Database: SQLite
- Maps: Leaflet.js
